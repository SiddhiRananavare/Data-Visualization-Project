# Data-Visualization-Project

-Created interactive and insightful data visualizations using Matplotlib and Seaborn to identify trends, patterns, and key insights.

📊 Employee Data Visualization Project

📌 Project Overview

-This project focuses on visualizing employee data to uncover patterns and insights using graphical representations. It helps in understanding salary distribution, department trends, and workforce structure through clear and interactive visuals.

🛠 Tools & Technologies

-Python

-Pandas

-NumPy

-Matplotlib

-Seaborn

📊 Steps Performed

-Data Collection & Loading.

-Data Cleaning & Preprocessing.

-Exploratory Data Analysis (EDA).

-Data Visualization using charts and graphs.

-Insight Generation from visual patterns.

📈 Visualizations Included

📌 Count Plot- For gropus of Employees.

📌 Bar Chart – Group-wise Average Salary.

📌 Histogram – Age and Salary distribution.

📌 Box Plot – Salary variation.

📌 Heatmap – Correlation analysis.

📌Scatter Plot- Healthy Eating vs Salary.


🎯 Result

-Identified salary distribution trends across employees.

-Compared employee count across departments.

-Visualized relationships between different features.

-Improved data understanding using graphical insights.

🔗 Dataset

-Employee dataset (can be extended using real-world datasets like Kaggle)
